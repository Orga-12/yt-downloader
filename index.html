⚙ Setup in 3 Schritten

1.Backend auf Railway.app hochladen: Neues Projekt → "Deploy from GitHub" → backend/ Ordner pushen → fertig.
2.Deine Railway-URL oben eintragen (z.B. https://yt-drive.up.railway.app) und auf Speichern klicken.
3.Für Google Drive: Google Cloud Console → OAuth 2.0 Client ID erstellen → Client ID in backend/main.py eintragen.

Diese HTML-Datei kannst du einfach lokal öffnen — kein Hosting nötig.
